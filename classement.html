⚙️ Comment fonctionne le classement ?

Le classement affiche les 25 meilleurs collectionneurs du serveur, triés selon deux critères. Utilise la commande /classement dans Discord pour voir le classement en temps réel.

🃏
Cartes uniques
Le nombre de cartes différentes dans ta collection. C'est le critère principal.
Critère n°1
📦
Total de cartes
Le nombre total de cartes possédées (doublons inclus). Sert de départage.
Critère n°2
